# Cadastro Empresarial

Projeto web de **Cadastro Empresarial** desenvolvido em **PHP** com **MySQL** e front-end em **HTML/CSS**. Permite o registro, edição e gerenciamento de empresas com validação de informações como CPF/CNPJ, e-mails e números de telefone. Inclui sistema de login e painel administrativo.

---

## Funcionalidades

- Cadastro de empresas com:
  - Nome da empresa
  - CPF ou CNPJ
  - E-mail válido
  - Telefone
  - Endereço
- Validação de dados antes do envio:
  - CPF ou CNPJ correto
  - E-mail válido
  - Telefone numérico
- Login para acesso ao painel administrativo
- Painel com opções de:
  - Visualizar empresas cadastradas
  - Editar dados de empresas
  - Deletar registros (se necessário)
- Estrutura modular usando **includes** para conexão com banco e cabeçalho/rodapé
